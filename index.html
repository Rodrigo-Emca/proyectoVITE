GRID primera Etapa

GRID es un sistema de diseño bidimensional en CSS que permite crear diseños complejos y estructurados en una pagina web. Es una forma mucho más poderosa de controlar la ubicación y el diseño de los elementos HTML.

A diferencia de flexbox, sistema unidimensional, CSS GRID ofrece una mayor flexibilidad y control sobre el diseño web.
